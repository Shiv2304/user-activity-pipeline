# User Activity Analytics Pipeline

Project scaffold for a Python-based user activity analytics pipeline.

## Structure

- `etl/`: extract, transform, and load modules.
- `sql/`: SQL queries used by the pipeline.
- `analytics/`: analytics and metric computation logic.
- `data/`: local data files (ignored except placeholders).
- `logs/`: runtime logs.
- `output/`: pipeline outputs (reports, exports, artifacts).
